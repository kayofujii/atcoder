# 入力コピペ用

(1)数字が1つ
入力例：N

N=int(input())

(2)数字が2つ以上で別々に受け取り入力例：AB

A,B=map(int,input().split())

(3)文字列が1つ入力例：S

S=input()

(4)文字列が2つ以上で別々に受け取り
入力例：ST

S,T=map(str,input().split())

(5)リストで受け取り
入力例：1 2 3

A=list(map(int,input().split()))
[1, 2, 3]

(6)二次元配列でうけとり
A = [list(map(int, input().split())) for _ in range(H)]

```
2 1 4
3 1 3
6 4 1
```

```
[[2, 1, 4], [3, 1, 3], [6, 4, 1]]
```

(7)数字を配列で受け取る 
入力例:1234
出力例:[1,2,3,4]
X = [int(i) for i in list(input())]
